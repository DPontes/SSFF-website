## Project Overview

This is the website for **Skårsskolans Föräldraförening (SSFF)** — the parent association at Skårsskolan. It is a simple static landing page in Swedish.

## Site Structure

Single-page layout with the following sections, in order:

1. **Hero / Introduction** — What SSFF is and what it does (content to be provided)
2. **Events** — Upcoming events organized by the association (content to be provided)
3. **Bli medlem (Become a member)** — Instructions for joining via Swish
4. **Kontakt (Contact)** — Contact information for the association

## Design Intent

- Simple, clean, and welcoming — aimed at parents of school-age children
- Content is in **Swedish**
- Mobile-friendly layout
